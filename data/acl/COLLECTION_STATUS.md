# ACL Paper Collection Status

This document tracks what papers have been collected.

## Collection Format

- **Year**: Conference year (e.g., 2024, 2023, 2022, 2021)
- **Conference**: ACL (Association for Computational Linguistics)
- **Track**: long, short
- **Paper Range**: Auto-detected by trying papers 1, 2, 3... until 404 error
- **Output Format**: CSV files named `YEAR-TRACK.csv` (e.g., `2024-long.csv`)
