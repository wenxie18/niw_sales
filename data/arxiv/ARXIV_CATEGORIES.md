## Usage Examples

Query specific category and year:
```bash
python 2.1-query_arxiv_papers.py --category eess.SY --year 2025 --max-results 1000 --output data/arxiv/eess_sy_2025.csv
```

Query multiple years for same category:
```bash
python 2.1-query_arxiv_papers.py --category cs.RO --year 2024 --max-results 5000
python 2.1-query_arxiv_papers.py --category cs.RO --year 2025 --max-results 5000
```
